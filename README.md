# Magic 8 Ball – College Edition

An interactive Magic 8 Ball web app that lets users ask a question about college life and receive a randomized response. After entering a question, users click the 8 ball, triggering a shake animation before revealing an answer related to classes, exams, internships, and student life.

This project was built as a fun way to explore JavaScript-driven interaction and animation while creating a playful user experience.

## Features
- Ask any college-related question
- Click the 8 ball to trigger a shake animation
- Randomized answer generation
- Smooth CSS animations for visual feedback
- Dynamic text updates using JavaScript

## Tech Stack
- **HTML** – Page structure and input elements
- **CSS** – Styling and shake animation
- **JavaScript** – Event handling, random answer selection, and DOM updates

## What I Practiced
- JavaScript event listeners
- Randomized response logic
- DOM manipulation
- CSS animation for interactive UI feedback
- Building a small, self-contained interactive web tool

## How It Works
1. User types a question into the input field.
2. The user clicks the Magic 8 Ball.
3. JavaScript triggers a CSS shake animation.
4. A randomized response is selected and displayed.
